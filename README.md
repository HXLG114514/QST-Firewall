这是一套具有自主监测防护能力的防火墙系统，可以有效杜绝恶意资源消耗（如挖矿，恶意消耗等）

1.0-Beta版本（版本号25829）：

·初具自主防护能力，可以有效杜绝恶意资源消耗（如挖矿，恶意消耗等）

·可以监测高频IP请求，并判断是否存在DDoS嫌疑，若存在，会自动拉黑IP

·监测资源高消耗进程，保护硬件资源

请注意：

·IP防护功能在该版本尚不成熟，请不要冒险抗压，后续版本会逐步完善

Beta仅供测试，并不具备较好的抗压能力，请谨慎使用
